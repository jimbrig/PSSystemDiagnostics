# Clear-NuGetCache

## SYNOPSIS
Clears the NuGet cache.

## SYNTAX

```
Clear-NuGetCache [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
This function clears the NuGet cache by enumerating the known caches and clearing them.

## EXAMPLES

### EXAMPLE 1
```
Clear-NuGetCache
Clears the NuGet cache.
```
